# Projeto Braço Robótico com Arduino

Este é um projeto prático de robótica onde desenvolvi um braço mecânico controlado por Arduino. O objetivo foi aplicar conhecimentos de lógica de programação (C++), eletrônica básica e controle de servomotores.

## 🤖 Demonstração
Confira o funcionamento do projeto no vídeo abaixo:
[Assista ao vídeo do braço em movimento](Braco%20Robotico.mp4)

## 🛠️ Tecnologias e Componentes
- *Hardware:* Arduino Uno, Servomotores (SG90/MG995), Jumper wires.
- *Software:* Arduino IDE (Linguagem C++).
- *Conceitos:* Controle via PWM (Pulse Width Modulation) para ajuste preciso de ângulos.

## 🚀 Como funciona
O código controla quatro servomotores independentes, permitindo movimentos articulados (Base, Braço 1, Braço 2 e Garra). As variáveis no código permitem ajustar o ângulo de abertura e fechamento para evitar que o braço force a estrutura mecânica.

## 💡 O que aprendi
- Montagem de circuitos com servomotores.
- Gerenciamento de múltiplos componentes eletrônicos via Arduino.
- Resolução de problemas relacionados à alimentação externa de motores.
